# DSA-in-Java
Data Structures & Algorithms implementations and problem-solving practice in Java.

# Java Linked List Implementation (MyLinkedList)

A custom implementation of a singly linked list in Java using OOP concepts.

## Features

- Add node at beginning
- Add node at end
- Add node at specific index
- Remove first node
- Remove last node
- Remove node at specific index
- Get first element
- Get last element
- Get element by index
- Search element using contains()
- Reverse linked list (Iterative)
- Reverse linked list (Recursive)
- Check size
- Check if list is empty
- Clear the list

## Concepts Used

- Object-Oriented Programming (OOP)
- Recursion
- Dynamic Data Structures
- Pointer Manipulation
- Encapsulation

## Project Structure

```text
LinkedList/
 └── MyLinkedList.java
```


# Java HashMap Implementation (MyHashMap)

A custom implementation of a HashMap in Java using separate chaining and rehashing.

## Features

- Put key-value pairs
- Get value using key
- Remove key-value pair
- Check key existence using containsKey()
- Get all keys using keySet()
- Handle collisions using separate chaining
- Automatic rehashing
- Generic implementation using `<K, V>`
- Custom toString() support

## Concepts Used

- Hashing
- Collision Handling
- Separate Chaining
- Rehashing
- Generics
- Load Factor / Lambda
- Time Complexity Optimization

## Project Structure

```text
HashMap/
 └── MyHashMap.java
```


# Java HashSet Implementation (MyHashSet)

A custom implementation of a HashSet in Java using separate chaining and dynamic rehashing.

## Features

- Add elements using add()
- Remove elements using remove()
- Check element existence using contains()
- Get total number of elements using size()
- Check if set is empty using isEmpty()
- Clear all elements using clear()
- Prevent duplicate elements
- Handle collisions using separate chaining
- Automatic rehashing based on load factor
- Generic implementation using `<K>`
- Custom Iterator support
- Enhanced for-each loop support
- Custom toString() support

## Concepts Used

- Hashing
- Collision Handling
- Separate Chaining
- Rehashing
- Generics
- Iterable Interface
- Iterator
- Load Factor
- Time Complexity Optimization

## Project Structure

```text
HashSet/
```
