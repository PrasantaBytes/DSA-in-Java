# Java Linked List Implementation(MyLinkedList)

A custom implementation of a singly linked list in Java using OOP concepts.

## Features

- Add node at beginning
- Add node at end
- Add node at specific index
- Remove first node
- Remove last node
- Remove node at specific index
- Get first element
- Get last element
- Get element by index
- Search element using contains()
- Reverse linked list (Iterative)
- Reverse linked list (Recursive)
- Check size
- Check if list is empty
- Clear the list

## Concepts Used

- Object-Oriented Programming (OOP)
- Recursion
- Dynamic Data Structures
- Pointer Manipulation
- Encapsulation
